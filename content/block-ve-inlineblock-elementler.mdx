<h3>Inline element</h3>

Aşağıdaki kodda her şey yan yana sıralanır. Buda inline element olduğunu gösterir.

``` html
lorem <a href="">link</a> <span>span</span> <b>bold</b> <i>italic</i>
```


<h3>Block element</h3>


``` html
<div>Div</div>
<p>Paragraf</p>
<h1>Başlık</h1>
```

Bu kodda html etiketleri  satır atlar.

<h3>İnline Block element</h3>

Block elemente width, height veremezsin ancak html etiketine inline-block özelliği eklersek width, height özelliklerini verebiliriz.
